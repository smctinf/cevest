Atualizações do Sistema

16/06/2020 Diversas alterações, como permitir emissão de pautas pelo próprio professor; consultar lista de celulares de alunos de uma turma pelos funcionários da recepção, afastamento do campo data de nascimento do campo quantidade de filhos, e mudança da tela de cadastro e alteração para o feio em Python.
22/01/2020 Implantada a opção de encerrar turma, que verificará se ainda há aluno com status CURSANDO. Só será possível emitir certificado para as turmas encerradas.
02/08/2019 Foi criada opção para que os instrutores possam listar alunos por turma com número de celular.
18/07/2019 Alteração na emissão do certificado. Agora, pode-se informar um CPF para que emita um certificado específico para um aluno. Se CPF não informado, serão emitidos todos os certificados para aquela turma selecionada.
11/07/2019 Ajustes no layout do certificado.
07/07/2019 Alteração na opção de Aprovação/Reprovação de aluno para que só apareçam as respectivas turmas de cada professor. No caso do usuário ser Administrador, verá todas as turmas.
