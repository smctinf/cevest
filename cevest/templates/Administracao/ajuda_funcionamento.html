Sistema Administrativo


Funcionamento do Sistema


- CADASTRO DOS ALUNOS -

O aluno fará seu cadastro, informando os cursos desejados e os turnos (manhã, tarde ou noite) que tenha disponibilidade.

Este cadastro ficará armazenado no sistema eternamente.

Conforme forem abrindo turmas (turmas previstas), o sistema buscará neste cadastro todos os alunos que demonstraram interesse no respectivo curso, e que tenha disponibilidade para o turno oferecido.

- TURMAS E SELEÇÃO DOS ALUNOS -

O administrador do sistema deverá criar uma TURMA PREVISTA, e então executar a alocação. Essa alocação selecionará os alunos que tenham marcado interesse no respectivo curso, além de disponibilidade para o turno oferecido.

Caso tenham mais alunos que vagas disponíveis, o módulo de alocação irá priorizar os alunos segundo os critérios apresentados no edital publicado.

Conforme os alunos forem sendo contactados, deverão ter sua situação alterada para "Desistiu da Vaga", "Não compareceu", ou clicar sobre o nome para confirmar o cadastro.

Ao confirmar o cadastro, o aluno terá sua situação alterada para "Matriculado".

Quando se decidir que uma TURMA PREVISTA será realmente oferecida, ou seja, quando quiser confirmar uma TURMA PREVISTA, o administrador do sistema deverá entrar na opção de Confirmar Turmas Previstas para efetivar esta confirmação.

A confirmação de uma TURMA PREVISTA criará um registro na tabela TURMA, e incluirá todos os alunos MATRICULADOS que estavam na relação de alunos da TURMA PREVISTA na relação da TURMA.

- CHAMAR MAIS ALUNOS PARA UMA TURMA PREVISTA -

Caso queira convocar mais alunos para completar uma turma prevista, o administrador deverá rodar novamente a alocação para a turma desejada.

A alocação poderá ser executada quantas vezes quiser. Assim, o sistema analisará quantas vagas faltam para completar a turma, e selecionará novos alunos.

Como o sistema verificará quantas vagas sobraram para chamar novos alunos, os candidatos que não compareceram ou abriram mão de suas vagas deverão ter seus status atualizados, sobrando assim vagas para novos candidatos.

- PAUTAS E CERTIFICADOS -

As pautas, lançamento de frequência e certificados só podem ser emitidos sobre turmas confirmadas.
